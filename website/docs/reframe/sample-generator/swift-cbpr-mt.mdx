## About This Generator

### Overview

The Swift CBPR+ MT Message sample generator provides a powerful and flexible way to create realistic test messages for various payment scenarios. The generator leverages predefined scenario configurations that encapsulate business rules, field mappings, and validation logic specific to each message type and use case. These scenario configs are implemented as JSON-based templates that define the structure, mandatory fields, optional fields, and business logic for generating compliant Swift MT messages according to CBPR+ standards.

### How It Works

The sample generator uses a sophisticated scenario configuration system where each message type (MT103, MT202, etc.) has multiple predefined scenarios representing real-world use cases. When you select a message type and scenario, the generator:

1. Loads the corresponding scenario configuration from the backend
2. Applies the business rules and field mappings defined in the config
3. Generates appropriate test data for variable fields
4. Validates the message structure against CBPR+ standards
5. Returns a fully-formed, compliant Swift MT message

The generator currently supports the following Swift MT message types:
